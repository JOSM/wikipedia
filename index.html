JOSM update site for dev-version of the wikipedia plugin

How to install the plugin site

Only do this, if you really need to try out the bleeding edge version of the wikipedia plugin. Otherwise you can install the latest released version like any other JOSM plugin.

Here is how you can install the current development version (not yet released, expect issues) of the wikipedia JOSM plugin:

  1. Open any JOSM instance in which you want to try out these development versions.
  2. Go to the preferences (e.g. hit F12).
  3. Make sure the checkbox Expert mode in the bottom left corner of the preferences dialog is checked.
  4. Go to the Plugins tab. Click on Configureâ€¦ in the bottom right corner.
  5. Then click on Add in the new dialog and enter the URL https://josm.gitlab.io/wikipedia/snapshot/master/update-site and confirm with OK.
  6. Now when you click Download list, you have a new plugin in the plugin list: wikipedia-dev. You can install that and it will be updated like the "normal" JOSM plugins.

Note: Please make sure that you don't have both the dev and the non-dev version (wikipedia and wikipedia-dev) installed at the same time! This would lead to conflicts.

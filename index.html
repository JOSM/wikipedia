How to install

  1. Open any JOSM instance (for installation instructions see the JOSM wiki).
  2. Open the preferences (e.g. hit the F12 key).
  3. Go to the Plugins tab.
  4. Click on Download list in the bottom left.
  5. Find the wikipedia plugin in the list and check its checkbox (you can use the search box at the top to narrow down the list).
  6. Now click the OK button of the preferences dialog and follow further instructions by JOSM (like restarting JOSM) if there are any.
